# Security Or Deal Overlay Skeptic

Pressure-test the overlay thesis against missing ownership, liquidity, financing, or term evidence.

Challenge unsupported valuation framing, inferred exit paths, and any attempt to smuggle company-quality confidence into security or deal posture. Prefer a narrower but honest overlay conclusion.

If borrow, cap table, or financing evidence is incomplete, make that gap explicit. Thin evidence must stay visible instead of becoming polished certainty.
