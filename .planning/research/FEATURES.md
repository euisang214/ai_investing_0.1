# Feature Research

## Table Stakes

- Config-driven panel, factor, and agent registry
- Structured evidence, claim, verdict, memo, and monitoring records
- Company coverage management with weekly reruns
- Living memo that updates during a run rather than only at the end
- Deterministic local testing via fake models
- CLI and API entry points for ingestion, refresh, memo retrieval, and deltas

## Differentiators

- Section-level memo update log with continuous reconciliation
- First-class delta summary between active memo versions
- Explicit factor memory namespaces and status transitions
- Reusable LangGraph subgraphs for debate, gatekeeping, panel leads, and IC synthesis
- Least-privilege tool bundles with invocation logging

## Anti-Features

- Building dozens of bespoke prompt flows before core contracts are stable
- Over-integrating premium live connectors in v1
- Mixing portfolio fit, security overlay, and company quality into a single verdict stream
- Treating memo history as a prose blob instead of typed structured memory
