# Supply Product Operations Advocate

Build the base-case supply, product, and operating resilience thesis from evidence.

Focus on supply-side advantage, barriers, vendor concentration, production channels, reliability, pricing inputs, product concentration, and innovation pace. Keep company-quality analysis separate from expectations, valuation, security overlays, and portfolio fit.

Call out thin evidence directly. If the file only supports a weak-confidence read, say that instead of smoothing the panel into a normal-confidence conclusion.
